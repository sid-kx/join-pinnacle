# Join Pinnacle Website

Recruitment landing page for **Pinnacle Realty, Brokerage** at **join.pinnaclerealty.ca**.

This repo contains the dedicated realtor recruitment website for Pinnacle Realty. The site is designed to attract ambitious Ontario real estate agents and guide them toward booking a confidential meeting with the broker.

## Live Site

**Production:** https://join.pinnaclerealty.ca

## Project Overview

The Join Pinnacle website is a focused recruitment funnel for real estate agents who are considering joining Pinnacle Realty.

The site communicates Pinnacle’s value proposition through a premium brokerage brand, clear commission messaging, agent support benefits, blog/article previews, and a direct application form.

## Main Purpose

The goal of this site is to convert interested realtors into meeting-request leads for the broker.

Unlike the private portal, this website does **not** create agent accounts. Submissions are treated as recruitment inquiries and are sent to the brokerage leadership team for follow-up.

## Target Audience

This website is built for:

- New real estate agents
- Experienced realtors
- Agents looking for stronger brokerage support
- Realtors interested in higher commission control
- Agents who want modern marketing and technology systems
- Residential and commercial real estate professionals in Ontario

## Important Workflow Note

This site is for **recruitment inquiries only**.

A form submission from join.pinnaclerealty.ca should:

1. Capture the agent’s contact information
2. Send the inquiry to the broker / leadership team
3. Trigger follow-up for a private conversation
4. Not automatically create an agent account
5. Not give portal access without broker approval

Actual agent accounts and profile management belong in the private portal.
